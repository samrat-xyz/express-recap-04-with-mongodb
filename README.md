# express-recap-04-with-mongodb

This is a simple practice project built with **Express.js** and **MongoDB** to recap basic **CRUD operations**.
The main goal of this project is practice and revising backend fundamentals.

---

##  Features

- Express server setup
- MongoDB connection using MongoDB Node.js Driver
- CRUD operations (Create, Read, Update, Delete)
- Uses environment variables with dotenv


---

##  Technologies Used

- Node.js
- Express.js
- MongoDB
- dotenv
- cors
